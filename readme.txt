HotSpotAI Steam 万能工具箱

== Description ==

本插件为HotSpotAI Steam提供了一系列实用的工具箱功能，包括搜索功能优化、内容审核等。

== Installation ==


1. 将`tool-box`文件夹上传到WordPress的`/wp-content/plugins/`目录。

2. 在WordPress后台的"插件"页面激活插件。

3. 根据需要配置插件选项。

== Frequently Asked Questions ==

= 插件如何启用搜索功能登录要求？ =
在插件的设置页面中，勾选“搜索功能登录要求”选项并保存设置。

= 插件如何实现内容审核功能？ =
接入第三方文本内容审核API，用户仅需填写相关内容即可
